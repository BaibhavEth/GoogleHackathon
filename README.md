## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
   - Optionally, set `FAL_API_KEY` for cost-effective image generation alternative
3. Run the app:
   `npm run dev`
